# Roadmap

## 🎯 Vision

Parchment aims to be the most refined, offline-first PDF reading experience on Android — purpose-built for religious and scholarly texts with bilingual support.

## 📅 Planned Milestones

### v1.1 — Enhanced Reader Experience
- [ ] Text search within PDF
- [ ] Adjustable font scaling / render quality
- [ ] Brightness slider (independent of system)
- [ ] Keep-screen-on toggle during reading
- [ ] Landscape mode optimization

### v1.2 — Multi-Book Library
- [ ] Home screen with book grid (activate existing `HomeScreen`)
- [ ] User-imported PDFs from device storage
- [ ] Recently read / reading progress tracking
- [ ] Book cover thumbnail caching

### v1.3 — Annotation & Notes
- [ ] Highlight pages with color markers
- [ ] Per-page notes
- [ ] Export bookmarks and notes

### v2.0 — Platform Maturity
- [ ] Room database for structured persistence
- [ ] DataStore migration from SharedPreferences
- [ ] Jetpack Navigation integration
- [ ] Tablet / foldable layout optimization
- [ ] Accessibility improvements (TalkBack, content descriptions)
- [ ] Localization (Urdu, Arabic UI)

### Future Considerations
- [ ] EPUB support
- [ ] Audio / recitation companion mode
- [ ] Cloud sync for bookmarks
- [ ] Widget for daily reading reminders
